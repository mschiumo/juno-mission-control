## Research Summary: OAuth Providers with Google/Facebook Out-of-the-Box

### 1. Auth0 (by Okta)
**Type:** Managed SaaS

**Pricing:**
- **Free:** $0/mo - Up to 25,000 MAU
  - Unlimited social connections (Google, Facebook, etc.)
  - Custom domain (requires CC verification)
  - 5 organizations
  - Basic attack protection
- **Essentials:** $35/mo - Up to 500 MAU base, scales with usage
- **Professional:** $240/mo - Up to 500 MAU base
- **Enterprise:** Custom pricing

**OAuth Capabilities:**
- ✅ Google, Facebook, Apple, GitHub, Twitter, Microsoft, and 50+ social providers
- ✅ Unlimited social connections on all plans
- ✅ Custom domain support
- ✅ Enterprise SSO (SAML/OIDC)
- ✅ MFA (TOTP, SMS, Email)
- ✅ Passwordless authentication
- ✅ Machine-to-Machine tokens

**Ease of Integration:** ⭐⭐⭐⭐⭐ (Excellent)
- Comprehensive SDKs (React, Vue, Angular, Next.js, mobile)
- Pre-built Universal Login pages
- Extensive documentation
- Large community

**Best For:** Enterprise applications, complex auth requirements, rapid deployment

---

### 2. Firebase Authentication
**Type:** Managed SaaS (Google Cloud)

**Pricing:**
- **Spark Plan (Free):** $0 - Up to 50,000 MAU for basic auth
- **Blaze Plan (Pay-as-you-go):** 
  - Free up to 50K MAU for email/password + social login
  - Phone Auth: Billed per SMS sent
  - SAML/OIDC: Free up to 50 MAU, then pay-per-use

**OAuth Capabilities:**
- ✅ Google (native integration)
- ✅ Facebook, Apple, Twitter, GitHub, Microsoft, Yahoo, Play Games, Game Center
- ✅ Anonymous auth
- ✅ Email/password with verification
- ✅ Phone authentication
- ✅ Custom auth tokens

**Ease of Integration:** ⭐⭐⭐⭐⭐ (Excellent)
- Extremely easy for Firebase/Google Cloud projects
- Native SDKs for all platforms
- Drop-in UI components
- Seamless integration with other Firebase services

**Best For:** Mobile apps, Firebase/GCP projects, startups, MVPs

---

### 3. Clerk
**Type:** Managed SaaS

**Pricing:**
- **Free:** $0/mo - Up to 50,000 Monthly Retained Users (MRU)
  - Unlimited social providers
  - 3 applications
  - Pre-built UI components
- **Pro:** Starting at usage-based pricing after free tier
  - $0.02/mo per MRU (50K-100K)
  - $0.018/mo per MRU (100K-1M)
  - $0.015/mo per MRU (1M-10M)

**OAuth Capabilities:**
- ✅ Google, Facebook, GitHub, GitLab, Discord, Twitter, Apple, Microsoft, and more
- ✅ Pre-built sign-in UI components
- ✅ Session management
- ✅ Organization/multi-tenancy support
- ✅ MFA
- ✅ RBAC
- ✅ Enterprise SSO

**Ease of Integration:** ⭐⭐⭐⭐⭐ (Excellent - especially for React/Next.js)
- Best-in-class React/Next.js SDK
- Beautiful pre-built components
- Middleware support
- Very developer-friendly

**Best For:** React/Next.js applications, modern web apps, SaaS products

---

### 4. Nango
**Type:** Unified API Platform (SaaS)

**Pricing:**
- **Free:** $0/mo
  - 10 API auth connections
  - 100k proxy requests
  - 10 hours function compute
  - 2 environments
- **Team:** Usage-based after free limits
  - $1 per additional connection after 20
  - $0.0001 per proxy request after 200k
  - Various usage tiers

**OAuth Capabilities:**
- ✅ 400+ pre-built API integrations
- ✅ Unified OAuth handling
- ✅ Token refresh management
- ✅ Two-way syncing
- ✅ Webhook handling
- ✅ API unification layer
- ⚠️ NOT a traditional auth provider - focuses on API integrations

**Ease of Integration:** ⭐⭐⭐⭐ (Very Good)
- Code-first approach
- Good for AI agents and integrations
- Pre-built connectors

**Best For:** Applications needing many third-party API integrations, AI agents, integration-heavy products

---

### 5. WorkOS
**Type:** Enterprise-focused Managed SaaS

**Pricing:**
- **AuthKit (User Management):** FREE up to 1 million MAU
  - Then $2,500/mo per additional million users
- **Enterprise SSO:** $125 per connection/month
- **Directory Sync:** $125 per connection/month

**OAuth Capabilities:**
- ✅ Social Auth (Google, Microsoft, etc.) - FREE
- ✅ Enterprise SSO (SAML, OIDC)
- ✅ Directory Sync (SCIM)
- ✅ MFA (TOTP, SMS)
- ✅ Passkeys/Biometric auth
- ✅ Magic Auth (passwordless)
- ✅ RBAC
- ✅ Admin Portal for self-service

**Ease of Integration:** ⭐⭐⭐⭐ (Very Good)
- Clean APIs
- Admin Portal included
- Good enterprise feature set
- Slightly more complex than Clerk/Auth0 for simple use cases

**Best For:** B2B SaaS, enterprise applications, companies selling to large organizations

---

  - Unlimited users
  - Social OAuth providers
  - 500MB database storage
- **Pro:** $25/mo
  - 8GB database
  - 100GB bandwidth
  - Priority support

**OAuth Capabilities:**
- ✅ Google, Facebook, GitHub, GitLab, Twitter, Discord, Apple, Azure, Bitbucket, Slack, and many more
- ✅ Email/password
- ✅ Magic links
- ✅ Phone auth
- ✅ SAML (enterprise add-on)

**Ease of Integration:** ⭐⭐⭐⭐ (Very Good)
- Great for existing Supabase users
- Good SDKs
- Open source (can self-host)

**Best For:** Supabase/Firebase alternatives, PostgreSQL-backed apps, open-source enthusiasts

### 7. Logto
**Type:** Open Source / Managed SaaS

**Pricing:**
- **Free (Self-hosted):** $0 - Unlimited users, full features
- **Cloud Free:** $0/mo - Up to 50,000 MAU, 50K tokens
- **Cloud Pro:** $24/mo - 50K tokens included, then usage-based
  - Enterprise SSO add-on: $48/mo
  - MFA add-on: $48/mo
- **Enterprise:** Custom pricing

**OAuth Capabilities:**
- ✅ Google, Facebook, GitHub, Discord, Apple, Microsoft, Azure AD, and more
- ✅ Multi-tenancy/Organizations
- ✅ RBAC
- ✅ Enterprise SSO (SAML/OIDC)
- ✅ MFA
- ✅ Machine-to-Machine auth
- ✅ Passkeys

**Ease of Integration:** ⭐⭐⭐⭐ (Very Good)
- Modern developer experience
- Good documentation
- Open source option
- Multi-tenancy features

**Best For:** SaaS applications, multi-tenant apps, cost-conscious teams, open-source preference

**Pricing:**
- **Community (Self-hosted):** FREE - Unlimited users, core features
- **Starter:** $125/mo - Premium features
- **Essentials:** $850/mo - Advanced features + support
- **Enterprise:** $3,300/mo - Full enterprise features
- **Cloud Hosting:** $37/mo (Basic) to $500/mo (HA)

**OAuth Capabilities:**
- ✅ Google, Facebook, Twitter, OIDC providers
- ✅ Unlimited social providers
- ✅ SAML
- ✅ MFA (TOTP, SMS, Email)
- ✅ Passkeys/WebAuthn
- ✅ Machine-to-Machine
- ✅ Breached password detection
- ✅ Gaming identity providers

**Ease of Integration:** ⭐⭐⭐ (Good)
- Self-hosted requires infrastructure knowledge
- Good SDKs and APIs
- Feature-rich but steeper learning curve

**Best For:** Enterprises wanting control, gaming applications, compliance-heavy industries

---

### 9. Keycloak
**Type:** Open Source (Self-hosted only)

**Pricing:**
- **FREE** - 100% open source, self-hosted

**OAuth Capabilities:**
- ✅ Google, Facebook, GitHub, and any OIDC/SAML provider
- ✅ Identity federation
- ✅ SSO
- ✅ MFA
- ✅ Fine-grained authorization
- ✅ User federation (LDAP, Active Directory)

**Ease of Integration:** ⭐⭐ (Fair)
- Requires self-hosting and maintenance
- Java-based
- Steeper learning curve
- No commercial support (unless via Red Hat)

**Best For:** Large enterprises with DevOps resources, government, highly regulated industries

## Recommendations

### For Quick Start / MVPs:
1. **Firebase Auth** - If already using Firebase/GCP
2. **Clerk** - For modern React/Next.js apps (best DX)
3. **Supabase Auth** - For PostgreSQL-based apps

### For B2B / Enterprise:
1. **WorkOS** - Best value for enterprise features (free up to 1M users)
2. **Auth0** - Most mature, comprehensive features
3. **Logto** - Cost-effective with good multi-tenancy

### For Cost-Conscious / Open Source:
1. **Keycloak** - Free but requires maintenance
2. **Logto (self-hosted)** - Modern alternative to Keycloak
3. **FusionAuth (Community)** - Feature-rich free option

### For Integration-Heavy Apps:
1. **Nango** - Best for connecting to 400+ APIs

---

## Quick Decision Guide

- **Building with React/Next.js?** → Clerk
- **Already on Firebase/GCP?** → Firebase Auth
- **Need enterprise SSO cheap?** → WorkOS
- **Want open source + modern?** → Logto
- **Selling to enterprises?** → WorkOS or Auth0
- **Multi-tenant SaaS?** → Logto or Clerk
- **Gaming app?** → FusionAuth
- **Maximum control + free?** → Keycloak (self-hosted)
