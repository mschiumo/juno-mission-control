### Required Environment Variables for Gmail
```
GOOGLE_CLIENT_ID=your_client_id
GOOGLE_CLIENT_SECRET=your_client_secret
GOOGLE_REFRESH_TOKEN=your_refresh_token
```

### API Usage
- `GET /api/gmail` - Fetch recent emails (default 10)
- `GET /api/gmail?unreadOnly=true` - Unread emails only
- `GET /api/gmail?label=SENT` - Filter by label
- `GET /api/gmail?maxResults=20` - Adjust results (max 50)
- `GET /api/gmail?refresh=true` - Bypass cache
