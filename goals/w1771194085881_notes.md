**Cost Comparison for $5,000 spend:**

## 5. Specific Pricing Numbers Summary

### Cheapest Options for Market Sentiment Screener:

**Twitter Data:**
- TwitterAPI.io: $0.15 per 1,000 tweets (pay-as-you-go)
- Official X API Basic: $200/month for 15,000 reads

**Financial News:**
- Finnhub: FREE (60 req/min) - personal use
- Alpha Vantage: FREE (25 req/day) - prototyping
- Polygon.io Stocks: $29/month - unlimited calls

**News Aggregation:**
- NewsAPI.org: FREE (100/day) - dev only
- NewsAPI.ai: $90/month (5K tokens)

### Recommended Budget Setup:
- **Development/Testing**: Finnhub (free) + Alpha Vantage (free) + TwitterAPI.io (pay-as-you-go)
- **Production MVP**: Finnhub (contact for commercial) + TwitterAPI.io ($50-100/month) + NewsAPI.ai ($90/month)
- **Scale**: Polygon.io ($29-199/month) + TwitterAPI.io (volume) + Apify (if scraping needed)

### Apify for Scraping (if needed):
- Starter Plan: $29/month + usage
- Good for scraping websites without APIs
- 1 CU = $0.30 (1 GB RAM per hour)
